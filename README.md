# Weasel-Game-Engine
**Proyecto final de carrera**  

**Requisitos:**
- Tener instalado cmake o alguna herramienta similar.
- Tener instalado OpenGL (Preferible última versión disponible).

**Como ejecutar:**  
Ejecutar el comando Make y luego main (WINDOWS)

**Librerias utilizadas:**
- GLFW3
- Glew
- stbImage
- Dear ImGui
- LUA
