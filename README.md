# Weasel-Game-Engine
**Final project of Computer Engineering degree**  

**Requirements:**
- Have installed make or a similar tool.
- Have installed OpenGL (latest version desirable).

**How to run:**  
Use the command make and then main (WINDOWS)

**Used libraries:**
- GLFW3
- Glew
- stbImage
- Dear ImGui
- LUA
- irrKlang
